## 📖 About

Road Wise Academy is a Botswana-based digital driving theory education platform dedicated to helping learners pass their **Matshwao (Class B)** theory test with confidence.

The platform provides interactive digital learning materials, mock examinations, instant feedback, and connects learners with trusted driving schools across Botswana.

**Slogan:** *"Study Smart. Pass with Confidence."*

## 🔧 Features

### Homepage
- Hero section with CTA buttons
- Interactive preview card (mini demo)
- Feature cards (5 key benefits)
- How It Works journey (4 steps)
- Partner school preview
- Testimonials
- CTA banner

### Demo Test
- 10 challenging theory questions
- Multiple choice with 4 options
- 10-minute timer with warning state
- Progress bar & question counter
- Instant feedback with explanations
- Results dashboard (score, accuracy, time)
- Keyboard navigation support

### Driving Schools Directory
- Dynamic JSON data loading
- Real-time search (name/location)
- Location filter (auto-populated)
- Professional school cards with:
  - Logo, name, location
  - Services tags
  - "Road Wise Verified" badge
  - Contact buttons

### Student Zone
- Gateway to Google Sites platform
- Feature highlights
- Prominent "Access Student Zone" button
- "Not Enrolled" section

### Theory Test Guide
- Two paths: Partner assistance vs. self-booking
- Requirements section
- Test day checklist
- Road Wise Academy tip box

### Contact
- WhatsApp contact card
- WhatsApp Community link
- Email contact
- Enquiry form (sends to WhatsApp)
- Social media links (Facebook & WhatsApp)

### 404 Error Page
- Custom branded error page
- Navigation links to key pages
- On-brand design
